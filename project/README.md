# Обработчик Изображений - React Приложение

Полнофункциональное веб-приложение для загрузки, просмотра и преобразования изображений между форматами PNG, JPG и собственным форматом GB7.

## Возможности

✅ **Загрузка и просмотр изображений**
- Поддержка форматов: PNG, JPG/JPEG, GB7
- Перетаскивание файлов (drag & drop)
- Клик по области для выбора файла

✅ **Просмотр на Canvas**
- Отрисовка изображений на HTML5 Canvas
- Масштабирование изображения в контейнер
- Пиксельная визуализация для высокой четкости

✅ **Экспорт в разные форматы**
- Сохранение в PNG (с поддержкой прозрачности)
- Сохранение в JPG (с качеством 90%)
- Сохранение в формат GB7

✅ **Информация об изображении**
- Отображение ширины и высоты в пикселях
- Показ глубины цвета (в битах)
- Информация о наличии прозрачности

## Формат GB7 (GrayBit-7)

GrayBit-7 - это формат изображений в оттенках серого без сжатия для образовательных целей.

### Структура файла GB7

**Заголовок (12 байт):**
- 0-3: Сигнатура файла: `0x47 0x42 0x37 0x1D` ("GB7·")
- 4: Версия (текущая: 0x01)
- 5: Флаг (бит 0 - маска, биты 1-7 - зарезервированы)
- 6-7: Ширина (16-бит, big-endian)
- 8-9: Высота (16-бит, big-endian)
- 10-11: Зарезервировано (0x0000)

**Данные изображения:**
- Каждый пиксель: 1 байт
  - Биты 0-6: Значение серого (0-127)
  - Бит 7: Бит маски (если включена)

## Установка и Запуск

### Требования
- Node.js 18+
- npm или yarn

### Шаги установки

```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр собранного проекта
npm run preview
```

Приложение будет доступно по адресу `http://localhost:5173/`

## Использование

### Загрузка изображения

1. **Клик на область загрузки** - откроется диалог выбора файла
2. **Drag & drop** - перетащите файл на область загрузки
3. **Выбор формата** - PNG, JPG или GB7

### Экспорт изображения

После загрузки изображения нажмите кнопку соответствующего формата:
- **⬇️ Скачать PNG** - сохранить с прозрачностью
- **⬇️ Скачать JPG** - сохранить без прозрачности
- **⬇️ Скачать GB7** - сохранить в формате GB7

## Технологии

- **React 18** - UI framework

- **Vite 4** - Build tool
- **HTML5 Canvas** - Отрисовка изображений
- **CSS3** - Стилизация

## Примечания

- Формат GB7 использует 7 бит для значения серого цвета
- При загрузке PNG сохраняется информация о прозрачности
- Максимальный размер изображения: 65535 x 65535 пикселей (ограничение 16-бит)
- Приложение полностью работает в браузере без серверной части
